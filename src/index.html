Introduction

In this tutorial you'll learn how to seamlessly create, deploy, and configure a decentralized webstie using GrapesJS, Fleek, GitHub, and Avvy Domains. Avvy Domains provides an address naming system on the Avalanche blockchain which allows you to link your .avax domain to IPFS-hosted content. This ensures your website remians censorship-resistant an decentralized. We'll help you set up an account with GrapesJS, and guide you through GitHub for a seamless deployment to Fleek. Finally, we'll configure your Avvy Domain to point uour IPFS content. By the end of this tutorial you'll have a fully functional decentralized website accessible via your .avax domain.

0.0

Prerequisites

Before starting, ensure you have the following:

  1. An Avvy domain registered (e.g., mydomain.avax)
    1. How To Register A Domain
  2. Basic HTML/CSS knowledge
  3. A Core wallet or another EVM-compatible wallet
1.0

Create a Website with GrapesJS

1.1

GrapesJS Introduction

GrapesJS is a web-based, open-source tool that simplifies website building with a drag-and-drop interface. It’s great for creating web pages visually without coding knowledge.

1.2

Set up GrapesJS

To start with GrapesJS:

  1. Go to grapesjs.com
  2. Click “Webpage Demo” to access the editor
  3. Design your website using drag-and-drop components
2.0

Upload Files to GitHub

2.1

Create a GitHub Account

If you don't have a GitHub account, you can create one:

  1. Go to github.com
  2. Sign up by following the instructions
2.2

Upload Files to Repository

Once you have a GitHub repository:

  1. Create a new repository for your website files
  2. Upload your HTML and CSS files into a folder named “src”
3.0

Upload to Fleek for IPFS Deployment

3.1

Connect GitHub to Fleek

To deploy your site to IPFS:

  1. Go to Fleek and create an account
  2. Connect your GitHub account and select your repository
  3. Deploy to IPFS through Fleek
4.0

Configure Your Avvy Domain

To link your Avvy domain to your IPFS site:

  1. Go to Avvy Domains
  2. Connect your wallet and select your domain
  3. Set up a CNAME record pointing to your IPFS hash
5.0

Deploy .avax Domain

After configuring your Avvy domain, verify the settings:

  1. Return to Fleek and confirm the CNAME settings
  2. Click "Verify" and Fleek will activate your site on the .avax domain
